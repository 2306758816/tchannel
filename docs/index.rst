TChannel
========

TChannel is a network multiplexing and framing protocol for RPC.

The protocol is currently implemented in:

 * `Go`_
 * `Node`_
 * `Python`_

.. _Go: http://godoc.org/github.com/uber/tchannel/golang
.. _Node: https://github.com/uber/tchannel
.. _Python: http://tchannel.readthedocs.org/projects/tchannel-python/en/latest/
